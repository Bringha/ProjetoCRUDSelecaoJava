Descriçao do Projeto

Criar um CRUD (Create, Read, Update, Delete) de pessoas/alunos no terminal usando Java:
    - Deve possuir as funcionalidades:
        - Criar pessoas ou alunos - ok
        - Mostrar todas as pessoas e alunos criadas (listar na tela) - ok
        - Atualizar dados de uma pessoa ou aluno - ok
        - Deletar uma pessoa ou aluno - ok
        - Encerrar o programa - ok
    - Informaçoes de uma pessoa/aluno - ok
        - Nome - ok
        - Telefone - ok
        - Data de Nascimento - ok
        - Data de cadastro da pessoa - ok
        - Data da última alteraçao - ok
        - Nota final no curso - ok
    - Caso o usuário informe a nota final do curso, deve ser criado um aluno. Caso nao seja informada, deve ser criada uma pessoa. - ok
    - Criar um menu interativo que o usuário possa usar para executar as opçoes acima. ok
    - O menu deve continuar sendo exibido ate que o usuário use a opçao de encerrar a aplicaçao. ok
    - O avaliado deve usar arrays para armazenar as informaçoes criadas. - ok
    - O avaliado deve usar herança para que um aluno aproveite todos os dados da pessoa. - ok
    - Todas as informaçoes ficam em memória. Nao é necessário cadastrar em banco de dados. Ou seja, quando encerrar a aplicaçao, perde-se todo o histórico. - ok
